Frequently Asked Questions

Got questions? We've got answers.

Is Personal Mode really free forever?

Yes! Personal Mode is completely free with no hidden costs or time limits. All core features — unlimited notes, graph visualization, task management, and wiki-links — are yours to use forever. We believe everyone deserves great note-taking tools.

What happens to my team's data if I cancel Pro?

You'll keep read-only access to all your team notes, and you can export everything as Markdown files at any time. Your data is never locked away. You just won't be able to use real-time collaboration features like chat, calls, and whiteboards.

What's the difference between Personal and Team Mode?

Personal Mode is 100% local — your notes are stored as plain Markdown files on your device, work offline, and never touch a server. Team Mode adds real-time collaboration with cloud sync, team chat, video calls, and shared whiteboards.

Can I use Conceptualize offline?

Absolutely! Personal Mode works entirely offline since everything is stored locally. Team Mode requires an internet connection for real-time features like chat and calls, but you can still access and edit cached notes when offline.

Where is my data stored?

In Personal Mode, all data stays on your local device as plain .md files in a folder you choose. In Team Mode, notes sync via Google Drive (your storage) or Firebase (secure cloud). You always have full control and can export anytime.

Do you have access to my notes?

No. In Personal Mode, we have zero access — everything is local. In Team Mode, if you use Google Drive, files stay in your Drive. If you use Firebase storage, your data is encrypted and we don't access or analyze your content. Your notes are yours.

Why does Windows show a security warning?

Windows SmartScreen shows warnings for new apps that haven't built up a reputation yet. This is normal for any new software. To proceed, click "More info" then "Run anyway". Conceptualize is safe, open-source friendly, and doesn't contain any malware.
